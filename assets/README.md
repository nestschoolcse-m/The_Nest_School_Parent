# NEST ERP Parent App - Assets

This directory should contain the following image assets:

## Required Files

1. **icon.png** (1024x1024)
   - App icon for iOS and Android
   - Should be square with no transparency

2. **splash-icon.png** (512x512)
   - Icon shown on splash screen
   - Can have transparent background

3. **adaptive-icon.png** (1024x1024)
   - Android adaptive icon foreground
   - Should have padding for safe area

4. **favicon.png** (48x48)
   - Web favicon

5. **notification-icon.png** (96x96)
   - Android notification icon
   - Should be white/transparent PNG

## Recommended Design

- Use school logo or education-themed icon
- Primary color: #2563eb (Blue)
- Keep icons simple and recognizable at small sizes
